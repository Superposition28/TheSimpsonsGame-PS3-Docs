SimGroup (.dat)

Defines groups of entities, their behaviors, and attributes.

Header

Offset Type Name Description
0x08 UInt32 Magic 0x53696D47 ("SimG").
0x0C UInt32 GUIDHash Hash of GUID.
0x10 UInt32 EntityCount Number of entities in the group.

Entity Entry

Type Name Description
UInt8 SpawnMask Spawn configuration mask.
Byte[16] GUID Entity GUID (looked up via GuidOffset in header).
UInt32 BehaviorID Behavior identifier.
Bitfield Attributes Bitfield defining which attributes are active. Attributes are 4-byte values (Int, Float, or Pointers).
